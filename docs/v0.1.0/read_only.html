Read-only

Daf.ReadOnly — Module

Read-only Daf storage format.

source
Daf.ReadOnly.ReadOnlyView — Type
struct ReadOnlyView <: DafReader ... end

A wrapper for any DafWriter data, protecting it against accidental modification. This isn't typically created manually; instead call read_only.

source
Daf.ReadOnly.read_only — Function
read_only(daf::DafReader)::ReadOnlyView

Wrap daf with a ReadOnlyView to protect it against accidental modification. If given a read-only daf, return it as-is.
